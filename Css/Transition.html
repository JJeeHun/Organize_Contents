transition

  1. width, height 속성에서 %단위는 적용안됨.
  2. px,vw,vh 단위 사용해야함.
  3. height으로 사용시 view port 단위이기 때문에 길이가 늘어남.
  4. 그것을 해결하기 위해 max-height 속성으로 사용
  5. max-height으로 사용시 transition의 max-height의 시작점 값이 0, 100vh일 경우 속도차이가 남.
    ex) 0 -> 100vh로 변환 시 , 100vh(100vh로 늘어난 box가 실제로 100vh만큼 늘어난게 아니기 때문에) -> 0으로 변환시
  6. 100vh이면 n초만큼 0 -> 100vh만큼 늘어남. , 현재 열린 px -> 0만큼 줄어들기 때문에 속도 차이가남.
  7. 열릴때와 닫힐때의 transition을 따로 적용해줘서 속도 개선해야함.
